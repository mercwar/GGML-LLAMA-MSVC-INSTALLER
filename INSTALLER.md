## Installer Overview

The FireGem installer:

- Installs CMake  
- Installs Ninja  
- Loads MSVC BuildTools  
- Clones llama.cpp  
- Configures CMake  
- Builds the engine  
- Runs tests  

---

## What the Installer Installs

- MSVC Build Tools  
- CMake  
- Ninja  
- Git  

---

## Running the Installer

Run:

```
GGML-LLAMA-MSVC-INSTALLER.bat
```

---

## Install Steps

After installation:

1. Ensure `llama.lib` and `ggml.lib` exist  
2. Ensure your GGUF model is placed  
3. Use Copilot to generate your assistant
