## System Requirements

- Windows 10/11  
- 64‑bit CPU  
- 4 GB RAM minimum  
- Admin access  

---

## Build Tools

Installer auto‑installs:

- MSVC Build Tools  
- CMake  
- Ninja  
- Git  

---

## Source Code

Installer clones:

```
llama.cpp
```

## Provided Libraries

- MSVC object files  
- C/C++ headers  
- Minimal engine tree

Works with:

- Copilot  
- Gemini  
- Claude  
- OpenAI models

- Engine is C + some C++  
- LLM generates glue code  
- GGUF requires GGML + LLAMA
